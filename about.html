Explorador de Países e Culturas

Sobre o Projeto

A aplicação permite explorar dados culturais, geográficos e políticos de países ao redor do mundo através da REST Countries API.

Objetivo do Projeto

Fornecer uma ferramenta interativa para consulta de países, incentivando:

  • Uso de APIs REST
  • Desenvolvimento de interfaces responsivas
  • Manipulação assíncrona de dados
  • Boas práticas de programação web

Funcionalidades

  • Busca de países por nome
  • Filtro por continente
  • Visualização detalhada de informações
  • Sistema de favoritos usando localStorage
  • Design moderno e responsivo
